Easy2Convert GIF to PNG
Version 2.8, Freeware
http://www.easy2convert.com/gif2png/

Introduction
------------
Easy2Convert GIF to PNG is a small freeware 
utility for converting Graphics Interchange Format files (.gif) 
to Portable Network Graphics files (.png) easily and quickly.

System requirements
-------------------
 * Windows 2000/XP/2003/Vista/7/8
 * less than 3 Mb of free space on hard disk
 
Installation
------------
Run an installer/unzip all files to any folder and run 
the executable file (usually, "gif2png.exe").

Getting started
---------------
1. Select a gif-file to convert
2. Select an output png-file
3. Click on the "Convert" button.
